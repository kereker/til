*単項プラス演算子* を知らなかったのでざっくりと調べた。  

### これのしくみ
`'b' + 'a' + + 'a' + 'a'`   
をざっくり分けると  
`('b') + ('a') + (+'a') + ('a')`  
こうなる。  

肝となるのは `(+'a')` 。
文頭の `+` は `単項プラス演算子` というもので、  
ざっくりいうと、文字列を数値へキャストするときに使えるもの。  
ただし、 `'a'` は数値にはなれないので、 `+ 'a'` の結果は `NaN` になる。  

で、これらを再度式として組み合わせると、  
`('b') + ('a') + (NaN) + ('a')`  
になって、文字列結合された結果が、 `baNaNa` になる。
(NaNがなぜ文字列として扱われるのかは不明…)  
`baNaNa` を `.toLowerCase()` して、最終結果は `banana` となる。
